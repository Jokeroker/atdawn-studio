Settings

Event setup

Room names may only contain letters, numbers and underscores. Both rooms share one password (required so guests can be moved between them).

Guest features & audio

Changes bake into the links — re-copy the guest invite and OBS links after saving. Guests already connected keep their old settings until they rejoin.

Broadcast & recording

Recordings save to this computer's downloads folder. Standby settings apply to newly copied OBS links.

Stream Deck & remote control

Drive the studio from a Stream Deck (free Bitfocus Companion app, "VDO.Ninja" module) or any HTTP request. Each room gets its own key — add them as two Companion instances. This control room must stay open for commands to work.
